# Phase 4 - Design & Refinement

## Goal
Polish the warning experience for accessibility, internationalization, layout resilience, and performance.

## Technical & Design Focus
- `chrome.i18n` message catalogs and locale fallback behavior.
- Keyboard navigation, screen-reader support, and RTL-safe layout.
- Banner visual hierarchy, contrast, and inclusive presentation checks.
- Debounced observation, short-circuiting on non-video pages, and bounded refresh behavior.
